Ayuda de MotoGP-Desktop

En esta parte vamos a ver que son cada uno de los elementos de el menu de navagecion

Inicio

Pagina principal de MotoGP-Desktop. Contiene un carrusel de imagenes del circuito Automotodrom Brno. Tambien contiene noticias relacionadas con MotoGP obtenidas de una API externa.

Piloto

Pagina que contiene informacion varia del piloto. Contiene tanto resultados, como biografia, e imagenes, videos y audios.

Ayuda

Pagina de ayuda del sitio web. Contiene una breve descripcion de cada pagina del sitio.

Circuito

Pagina con informacion del circuito Automotodrom Brno. Permite cargar el archivo html para ver los datos del circuito, Cargar un svg para ver la altimetria y, cuenta con un mapa dinamico desde el que puedes cargar un kml para que te lleve alli.

Clasificaciones

Pagina que contiene clasificaciones de la carrera.

Juegos

Pagina con juegos interactivos sobre motogp. Contiene el juego de memoria, que consiste en encontrar parejas en cartas boca abajo.

Meteorología

Pagina que contiene informacion y meteorologia de la ciudad Brno. Esta ciudad es la mas proxima al circuito y su meteorologia coincide con la de este. Consulta a la API de Open-Meteo para obtener la informacion meteorologica.
